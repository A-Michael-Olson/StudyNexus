# Project Title
StudyNexus

## Team Members
 James Stolte, Mike Olson, Kaitlin Heusner, Joshua Moncada, Graham McClure
 
## Software Description
A group study planner/collaboration website that invites people working together to have
a shared space where people can work together, keep each other accountable, and add
tasks. This will include having a group where people can communicate as well as add, 
remove, and manage tasks. File sharing will be available within these groups. Also, there will
be a space for people to work together with a timer and chat. 

## Architecture
The architecture that we are going to use is: HTML, CSS, JavaScript, Visual Studio Code and GitHub.
